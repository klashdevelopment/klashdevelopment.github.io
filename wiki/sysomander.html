Sysomander is a spigot api for making commands.


Links: Setup Adding to your code Making a sysomand

Setup
  • See the link below
  • spigotmc.org/resources/sysomander-api-the-best-command-system-1-13-1-19.99863


Adding to your code
  • Make a sysomander instance in your main plugin file: public Sysomander commands;

  • In your onEnable() method: commands = new Sysomander(this);

  • In your onEnable() method aswell, initialize your commands like this: commands.addSysomandToServer(new Sysomand());

  • In your onCommand() method, set the return to: commands.runCommandSystem(sender, label, args);
